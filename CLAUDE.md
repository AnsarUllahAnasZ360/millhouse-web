# Millhouse Web

> Everything's coming up Millhouse

Real-time collaborative productivity platform with boards, notes, and todos.

## Stack
Next.js 16, React 19, TypeScript, Convex, Tailwind CSS 4, shadcn/ui (new-york), Bun

## Commands
- `bun run dev` - Development
- `bun run build` - Production build
- `bun run test` - Unit tests
- `bun run test:e2e` - E2E tests
- `bun run lint` - Linting
- `bun run typecheck` - Type check
- `npx convex dev` - Convex server

## Structure
- `src/app/` - Next.js pages
- `src/components/` - React components
- `src/components/ui/` - shadcn/ui (new-york style, neutral, inter font)
- `convex/` - Backend functions
- `tests/` - Test files
